# FurBot - NullPointer

## 简介

NullPointer 是一个基于 [nonebot](https://github.com/nonebot/nonebot2) 的 Furry 类 QQ Bot。 

目前正在计划重构并迁移至 [Mirai](https://github.com/mamoe/mirai)。
